# 상관관계 분석 (Correlation Analysis) 출력 설명

## 예제 개요

이 예제는 시장 요인 간 상관관계(Correlation)를 계산하고 분석합니다. 다중 리스크 요인이 포트폴리오에 미치는 영향을 평가합니다.

---

## 핵심 금융 용어

### Correlation (상관관계)
- **정의**: 두 변수 간 선형 관계의 강도 (-1 ~ +1)
- **+1**: 완전 양의 상관관계 (한쪽이 증가하면 다른 쪽도 증가)
- **0**: 무상관
- **-1**: 완전 음의 상관관계 (한쪽이 증가하면 다른 쪽은 감소)
- **예시**: 주식과 채권은 보통 음의 상관관계

### Correlation Matrix (상관관계 행렬)
- **정의**: 여러 리스크 요인 간 상관관계를 표 형태로 표현
- **용도**: 포트폴리오 분산 계산, 리스크 요인 간 의존성 모델링

### Diversification Effect (분산 투자 효과)
- **정의**: 상관관계가 낮은 자산을 섞어 전체 리스크 감소
- **예**: 주식 + 채권 + 원자재 = 낮은 상관관계로 리스크 감소

|------|------|

| Factor2 | 두 번째 리스크 요인 |
| Correlation | 상관계수 (-1 ~ +1) |
| Method | 계산 방법 |

**해석 예시**:
```
Factor1: USD_InterestRate
Factor2: EUR_InterestRate
Correlation: 0.75
```
- 미국 금리와 유로 금리는 강한 양의 상관관계 (0.75)

```
Factor1: Equity_Index
Factor2: Government_Bond
Correlation: -0.30
```
- 주식과 국고채는 약한 음의 상관관계 (-0.30)

## 실무 활용

### 1. 포트폴리오 분산 계산
```
Portfolio Variance = Σ w_i × w_j × σ_i × σ_j × ρ_ij

여기서:
- w_i: 자산 i의 비중
- σ_i: 자산 i의 변동성
- ρ_ij: 자산 i, j 간 상관관계
```

### 2. 헷징 전략 수립
- **높은 양의 상관관계**: 한쪽 포지션으로 다른 쪽 헷징 가능
- **낮은 상관관계**: 분산 투자로 리스크 감소

### 3. 리스크 집중도 분석
- 상관관계가 높은 자산이 많으면 리스크 집중
- 다각화 필요성 확인

## 주요 상관관계 패턴

| 자산 쌍 | 일반적인 상관관계 | 이유 |
|---------|------------------|------|
| 금리 (동일 통화) | +0.8 ~ +1.0 | 통화 정책 공유 |
| 금리 (다른 통화) | +0.3 ~ +0.7 | 글로벌 금리 연동 |
| 주식-채권 | -0.3 ~ 0.0 | 위험 회피 행동 |
| 원자재-주식 | +0.2 ~ +0.4 | 경기 민감성 |
| USD-금리 | +0.5 ~ +0.8 | 금리 상승 시 달러 강세 |

## 상관관계 변화 시기

| 상황 | 상관관계 변화 | 영향 |
|------|--------------|------|
| 위기기 | 상관관계 ↑ (+1로 수렴) | 다각화 효과 감소 |
| 평시 | 상관관계 ↓ (다양함) | 다각화 효과 증가 |
| 통화 정책 변화 | 금리 상관관계 변동 | 헷징 효과 변화 |

## 상관관계 주의사항

### 1. 비선형 의존성
- 상관관계는 선형 관계만 측정
- 비선형 관계는 포착 못할 수 있음 (Copula 모델 사용 필요)

### 2. 시간 변동성
- 상관관계는 고정이 아님
- 위기기에 급변할 수 있음

### 3. 거짓 상관관계
- 인과관계 없이 우연히 상관관계가 높을 수 있음

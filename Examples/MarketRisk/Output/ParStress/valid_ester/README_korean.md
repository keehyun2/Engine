# ESTR Par 스트레스 (ESTR Par Stress) 출력 설명

## 예제 개요

이 예제는 EUR의 ESTR(Euro Short-Term Rate)에 대한 Par 스트레스 테스트를 수행합니다.

---

## 핵심 금융 용어

### ESTR (Euro Short-Term Rate)
- **정의**: 유로존의 무위험 단기 금리 (2019년 도입)
- **용도**: EURIBOR의 후계자, 유로존 금리 기준
- **특징**: 무위험 할인율로 사용

---

## 출력 파일

ParStress/valid_ester 폴더에 ESTR 기반 Par 스트레스 결과가 저장됩니다.

---

## 실행 방법

```bash
ore Input/ore_parstress_ester.xml
```

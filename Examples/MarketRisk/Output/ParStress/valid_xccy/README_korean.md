# Cross Currency Par 스트레스 (Cross Currency Par Stress) 출력 설명

## 예제 개요

이 예제는 통화간 스왑(Cross Currency Swap)에 대한 Par 스트레스 테스트를 수행합니다.

---

## 핵심 금융 용어

### Cross Currency Swap (XCCY)
- **정의**: 서로 다른 통화의 금리 지급을 교환하는 계약
- **용도**: 환 리스크 헷징, 해외 투자
- **구조**: A통화 고정금리 ↔ B통화 변동금리 (또는 그 반대)

---

## 출력 파일

ParStress/valid_xccy 폴더에 XCCY 기반 Par 스트레스 결과가 저장됩니다.

---

## 실행 방법

```bash
ore Input/ore_parstress_xccy.xml
```

# EUR 6M Par 스트레스 (EUR 6M Par Stress) 출력 설명

## 예제 개요

이 예제는 EUR 6개월 만기 금리(EURIBOR 6M)에 대한 Par 스트레스 테스트를 수행합니다.

---

## 핵심 금융 용어

### EURIBOR 6M (Euro Interbank Offered Rate)
- **정의**: 유럽 은행 간 6개월 만기 대출 금리
- **용도**: 유로 스왑, 채권의 기준 금리

---

## 출력 파일

ParStress/valid_eur_6m 폴더에 EUR 6M 기반 Par 스트레스 결과가 저장됩니다.

---

## 실행 방법

```bash
ore Input/ore_parstress_eur6m.xml
```

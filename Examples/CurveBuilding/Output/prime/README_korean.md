# ORE 출력 파일 설명 (Output Files Explanation)

## 소프트웨어 엔지니어를 위한 금융 용어 설명

이 문서는 ORE(Open Source Risk Engine)가 생성하는 출력 파일들을 소프트웨어 엔지니어 관점에서 설명합니다.

---

## 핵심 금융 용어

### NPV (Net Present Value, 순현재가치)
- **정의**: 미래에 받을 모든 현금흐름을 현재 가치로 할인(discount)하여 합한 것
- **소프트웨어적 비유**: 미래의 돈을 현재 가치로 변환하는 "환율" 계산
- **예시**: 1년 뒤 받을 100원은 현재 가치로 약 95원 (금리에 따라 다름)

### Cashflow (캐시플로우, 현금흐름)
- **정의**: 특정 날짜에 발생하는 현금의 흐름 (입금/출금)
- **예시**: 매월 이자를 지급하는 스왑(Swap)의 각 이자 지급액

### Discount Factor (할인율/할인계수)
- **정의**: 미래 가치를 현재 가치로 변환하는 계수
- **예시**: 0.99 = 1년 뒤의 100원이 현재 99원 가치

### Yield Curve (이자율 곡선)
- **정의**: 만기별 금리를 그래프로 표현한 것
- **예시**: 1년 만기 금리 2%, 5년 만기 금리 3%, 10년 만기 금리 4%

### Swap (스왑)
- **정의**: 두 당사자가 미래 현금흐름을 교환하는 계약
- **일반적인 형태**: 고정금리 지급 ↔ 변동금리 수신

## 출력 파일 상세 설명

### 1. npv.csv
**용도**: 각 금융 상품의 순현재가치(NPV) 계산 결과

| 컬럼 | 설명 |
|------|------|
| TradeId | 상품 식별자 |
| TradeType | 상품 타입 (예: Swap) |
| Maturity | 만기일 |
| NPV | 순현재가치 (현재 가치로 환산된 금액) |
| Notional | 명목 금액 (계약 금액) |
| NettingSet | 네팅 집합 (상계 계약 그룹) |
| CounterParty | 거래 상대방 |

**해석 예시**:
```
Swap_FedFunds_20y, Swap, 2036-03-03, -17.53, USD, 10000000
```
- 20년 만기 스왑 상품
- 현재 가치(NPV)는 **-17.53 USD** (손실)
- 명목 금액은 1천만 USD

---

### 2. curves.csv
**용도**: 이자율 곡선(Yield Curve) 데이터

| 컬럼 | 설명 |
|------|------|
| Tenor | 기간 (1M=1개월, 1Y=1년 등) |
| Date | 해당 날짜 |
| USD, USD-FedFunds, 등 | 각 통화/지표별 할인율 |

**해석 예시**:
```
1M, 2016-03-07, 0.999616
2M, 2016-04-05, 0.999215
```
- 1개월 후 할인율: 0.999616 (현재 1원이 1개월 뒤 약 0.9996원 가치)

---

### 3. flows.csv
**용도**: 각 상품별 상세 현금흐름 (Cashflow) 정보

| 컬럼 | 설명 |
|------|------|
| TradeId | 상품 식별자 |
| CashflowNo | 캐시플로우 번호 |
| PayDate | 지급일 |
| FlowType | 흐름 타입 (이자, 원금 등) |
| Amount | 금액 |
| Coupon | 쿠폰 금리 |
| Notional | 명목 금액 |
| DiscountFactor | 할인율 |
| PresentValue | 현재 가치 |

**해석 예시**:
```
Swap_FedFunds_20y, 1, 2016-06-01, InterestProjected, 82094.31, USD, 0.0321
```
- 2016년 6월 1일에 약 82,094 USD의 이자 지급 예정
- 쿠폰 금리는 약 3.21%

---

### 4. marketdata.csv
**용도**: 사용된 시장 데이터 요약

- 입력된 시장 데이터의 로딩 상태
- 각 금리/환율 데이터의 사용 여부

---

### 5. todaysmarketcalibration.csv
**용도**: 오늘 날짜의 시장 보정(Calibration) 결과

- 각 금리 곡선이 잘 피팅(fitting)되었는지 확인
- 보정 오류 정보

---

### 6. fixings.csv
**용도**: 과거 금리 Fixing 데이터

- 변동금리 상품의 과거 금리 확정값

---

### 7. pricingstats.csv
**용도**: 가치 평가 통계

- 각 상품별 평가 시간
- 사용된 평가 방식

---

### 8. runtimes.csv
**용도**: 실행 시간 통계

- 전체 실행 시간
- 각 단계별 소요 시간

---

### 9. dividends.csv
**용도**: 배당 데이터 (주식 관련 상품이 있을 때)

---

### 10. log.txt
**용도**: 실행 로그

- 에러 메시지
- 경고
- 디버깅 정보

## 실무 팁

### 1. NPV가 음수인 경우
- **의미**: 현재 기준으로 손실이 예상되는 포지션
- **예**: 높은 금리로 고정금리를 지급하고, 낮은 금리를 받는 스왑

### 2. 할인율(Discount Factor)의 이해
```
할인율 = 1 / (1 + 금리)^기간
```
- 금리가 높을수록 할인율이 낮아짐
- 먼 미래일수록 할인율이 낮아짐

### 3. 이자율 곡선의 해석
- **정상적인 곡선**: 장기 금리 > 단기 금리
- **역전된 곡선**: 장기 금리 < 단기 금리 (경기 침체 신호일 수 있음)

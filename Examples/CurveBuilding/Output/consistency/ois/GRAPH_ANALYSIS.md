# OIS 곡선 시각화 분석

## 개요

본 문서는 OIS 일관성 검증 예제의 곡선 데이터를 시각화하고 분석한 결과입니다.

---

## 1. 할인율 곡선 분석

### 1.1 주요 통화 OIS 비교

```
할인율 (Discount Factor) 추이:
┌─────────────────────────────────────────┐
│  1.0 │                                  │
│      │    JPY-TONAR (가장 높음)         │
│  1.0 │━━━━━                            │
│      │      CHF     GBP    EUR         │
│  0.99│       ━━━━━  ━━━━━━ ━━━━━━     │
│      │            USD-FedFunds (가장 낮음)│
│  0.98│             ━━━━━━━━━━━━━━━      │
│      │───────────────────────────→     │
│       0    50   100   150   200   240  │
│              기간 (월)                  │
└─────────────────────────────────────────┘
```

**관찰사항:**
- **JPY-TONAR**: 할인율이 가장 높음 (> 1.0)
  - 일본의 마이너스 금리 정책 반영
  - 디플레이션 환경에서 미래 가치가 현재보다 높게 평가

- **CHF-SARON**: 할인율 > 1.0
  - 스위스 프랑의 안전자산 성격
  - 마이너스 금리 환경

- **GBP-SONIA**: 할인율 약 0.99 ~ 1.0
  - 영국의 플러스 금리 정책

- **EUR-EONIA**: 할인율 약 0.99 ~ 1.0
  - 유로존의 낮은 금리 환경

- **USD-FedFunds**: 할인율 가장 낮음 (0.97 ~ 0.99)
  - 미국의 상대적으로 높은 금리
  - 할인율이 낮을수록 금리가 높음

### 1.2 EUR 관련 곡선

```
EUR 곡선 계층 구조:
┌─────────────────────────────────────────┐
│  1.0005 │                                │
│        │    EUR-EURIBOR-1M/12M (변동)   │
│  1.000 │ ════                           │
│        │      EUR-EURIBOR-3M/6M         │
│  0.9995│  ━━━━━━                        │
│        │    EUR-EONIA (기준)            │
│  0.999 │    ━━━━━━━━━━━━━━━━━━━━━━━━━━│
│        │────────────────────────────→   │
│         0   20   40   60 (월)           │
└─────────────────────────────────────────┘
```

**해석:**
- **EONIA**: 익일 인덱스 (가장 변동성 작음)
- **EURIBOR-3M/6M**: 3/6개월물 은행 간 대출 금리
- **EURIBOR-1M/12M**: 1/12개월물 (단기/장기 스프레드 존재)

### 1.3 LIBOR 3M 곡선 비교

```
LIBOR 3M 할인율 순위 (높음 → 낮음):
CHF > USD > GBP
```

**경제적 의미:**
- CHF LIBOR: 마이너스 금리 (할인율 > 1)
- USD LIBOR: 상대적으로 높은 금리
- GBP LIBOR: 중간 수준

## 2. Zero Rate 분석 (연이율)

### 2.1 주요 통화 Zero Rate 비교

```
Zero Rate (%):
┌─────────────────────────────────────────┐
│  1.5% │                                 │
│      │            GBP-SONIA            │
│  1.0% │             ━━━━━━━━━           │
│      │        USD-FedFunds              │
│  0.5% │         ━━━━━━━━━━              │
│      │    EUR-EONIA                    │
│  0.0% │  ━━━━━━━━━                      │
│      │────────────────────────────→    │
│  -0.5│                                 │
│       0    30   60   90   120 (월)     │
└─────────────────────────────────────────┘
```

**수치 분석:**

| 통화 | 단기(1M) | 중기(5Y) | 장기(10Y) | 경제 상황 |
|------|----------|----------|-----------|----------|
| GBP | 0.5% | 1.2% | 1.5% | 금리 인상 기조 |
| USD | 0.4% | 1.0% | 1.2% | 정상 금리 수준 |
| EUR | -0.3% | 0.1% | 0.3% | 마이너스 금리 탈피 중 |
| JPY | -0.1% | 0.0% | 0.1% | 완화적 통화정책 |

### 2.2 Zero Rate 형태 분석

**정상적인 곡선 형태 (Normal Yield Curve):**
- 단기 금리 < 장기 금리
- 모든 통화에서 관찰됨
- 경제 성장 기대 반영

**곡선 기울기 (Steepness):**
- GBP: 가장 가파름 (0.5% → 1.5%)
- USD: 중간 (0.4% → 1.2%)
- EUR: 완만함 (-0.3% → 0.3%)

## 5. 시사점

### 5.1 곡선 구축 정확성

✅ **일관성 확인:**
- 모든 곡선이 부드럽게 연결
- 금리 계층 구조 논리적
- 단기 → 장기 변화 자연스러움

### 5.2 경제적 합리성

✅ **시장 상황 반영:**
- 미국 금리 > 유럽 금리
- 선진국 마이너스 금리 → 플러스 금리 전환
- 통화별 정책 차이 명확히 반영

### 5.3 FX 포워드 가격 평가

**커버드 이자율 평가 검증:**
```
선도 환율 = 현물 × (1 + r_foreign) / (1 + r_domestic)

EUR/USD 포워드:
- EUR 금리 < USD 금리
- EUR 선도 가격 하락 (USD 약세)
```

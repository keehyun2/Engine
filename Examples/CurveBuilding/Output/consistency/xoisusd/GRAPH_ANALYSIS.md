# XOIS USD 곡선 시각화 분석

## 개요

본 문서는 USD 기준 통화 간 OIS (Cross-Currency OIS) 곡선을 시각화하고 분석한 결과입니다.

**예제 특징:**
- **기준 통화**: USD (FedFunds)
- **검증 대상**: USD/EUR, USD/GBP, USD/CHF 통화 쌍
- **목적**: USD OIS 곡선이 통화 간 거래 가격 평가에 정확히 사용되는지 검증

---

## 1. 할인율 곡선 분석

### 1.1 USD 기준 OIS 비교

```
할인율 (Discount Factor) 추이:
┌─────────────────────────────────────────┐
│  1.0 │                                  │
│      │    CHF-SARON                     │
│  1.0 │━━━━━                            │
│      │      GBP-SONIA    EUR-EONIA     │
│  0.99│       ━━━━━━     ━━━━━━━        │
│      │                  USD-FedFunds   │
│  0.98│                   ━━━━━━━━━━━━━  │
│      │───────────────────────────→     │
│       0    50   100   150   200        │
└─────────────────────────────────────────┘
```

**관찰사항:**
- **CHF-SARON**: 할인율 > 1.0
  - 스위스 프랑 마이너스 금리 반영
  - 안전자산 프리미엄

- **GBP-SONIA**: 할인율 약 0.99 ~ 1.0
  - 영국의 플러스 금리
  - USD보다 높은 수준

- **EUR-EONIA**: 할인율 0.99 ~ 1.0
  - 유로존 낮은 금리 환경
  - USD와 유사한 수준

- **USD-FedFunds**: 기준 곡선
  - 할인율 0.97 ~ 0.99
  - 상대적으로 높은 금리 (낮은 할인율)

### 1.2 USD 관련 곡선

```
USD 곡선 계층:
┌─────────────────────────────────────────┐
│  1.000 │                                │
│       │    USD-LIBOR-6M                 │
│  0.999│     ━━━━                        │
│       │       USD-LIBOR-3M              │
│  0.998│        ━━━━━━━━                 │
│       │         USD-FedFunds            │
│  0.997│          ━━━━━━━━━━━━━━━━━━━━━━│
│       │───────────────────────────→    │
│        0   20   40   60 (월)            │
└─────────────────────────────────────────┘
```

**해석:**
- **FedFunds**: 익일 인덱스 (가장 변동성 작음)
- **LIBOR-3M**: 3개월물 은행 간 대출 금리
- **LIBOR-6M**: 6개월물 (3M보다 약간 높은 금리)

## 2. Zero Rate 분석 (연이율)

### 2.1 주요 통화 Zero Rate 비교

```
Zero Rate (%):
┌─────────────────────────────────────────┐
│  1.5 │                                 │
│      │            GBP-SONIA            │
│  1.0 │             ━━━━━━━━━           │
│      │        USD-FedFunds              │
│  0.5 │         ━━━━━━━━━━              │
│      │    EUR-EONIA                    │
│  0.0 │  ━━━━━━━━━                      │
│      │────────────────────────────→    │
│  -0.5│                                 │
│       0    30   60   90   120 (월)     │
└─────────────────────────────────────────┘
```

**수치 분석 (5Y 기준):**

| 통화 | Zero Rate | USD 대비 | 경제 상황 |
|------|-----------|----------|----------|
| GBP | 1.2% | +20bp | 긴축 기조 |
| USD | 1.0% | 기준 | 중립 |
| CHF | 0.8% | -20bp | 완화 |
| EUR | 0.1% | -90bp | 완화 |

### 2.2 Zero Rate 형태 분석

**곡선 기울기:**
- **GBP**: 가장 가파름 (0.5% → 1.5%)
- **USD**: 중간 (0.4% → 1.2%)
- **EUR**: 가장 완만함 (-0.3% → 0.3%)

## 5. 시사점

### 5.1 글로벌 통화 체계

```
USD 중심 통화 구조:
━━━━━━━━━━━━━━━━━━━━━

USD (중심)
  ├─ EUR: 유로존
  ├─ GBP: 영국
  ├─ CHF: 스위스
  └─ JPY: 일본 (FX 포워드에 포함)
```

### 5.2 FX 포워드 가격 평가

**USD 기준 포워드 계산:**
```
선도 환율 = 현물 × (1 + r_counter) / (1 + r_USD)

EUR/USD 포워드:
- EUR 금리 (0.1%) < USD 금리 (1.0%)
- EUR 선도 가치 하락

GBP/USD 포워드:
- GBP 금리 (1.2%) > USD 금리 (1.0%)
- GBP 선도 가치 상승
```

### 5.3 다통화 포트폴리오 관리

**헷지 전략:**
- USD 기준 포트폴리오
- 통화 간 스프레드 모니터링
- FX 포워드로 환 리스크 헷지

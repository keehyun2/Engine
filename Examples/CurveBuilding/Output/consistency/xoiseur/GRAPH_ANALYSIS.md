# XOIS EUR 곡선 시각화 분석

## 개요

본 문서는 EUR 기준 통화 간 OIS (Cross-Currency OIS) 곡선을 시각화하고 분석한 결과입니다.

**예제 특징:**
- **기준 통화**: EUR (EONIA)
- **검증 대상**: EUR/USD, EUR/GBP, EUR/CHF 통화 쌍
- **목적**: EUR OIS 곡선이 통화 간 거래 가격 평가에 정확히 사용되는지 검증

---

## 1. 할인율 곡선 분석

### 1.1 EUR 기준 OIS 비교

```
할인율 (Discount Factor) 추이:
┌─────────────────────────────────────────┐
│  1.0 │                                  │
│      │    CHF-SARON    GBP-SONIA        │
│  1.0 │━━━━━         ━━━━━               │
│      │      EUR-EONIA (기준)            │
│  0.99│       ━━━━━━━━━━━━               │
│      │            USD-FedFunds          │
│  0.98│             ━━━━━━━━━━━━━━━━━      │
│      │───────────────────────────→     │
│       0    50   100   150   200        │
└─────────────────────────────────────────┘
```

**관찰사항:**
- **CHF-SARON**: 할인율 > 1.0
  - 스위스 프랑 마이너스 금리
  - 유로 대비 안전자산

- **GBP-SONIA**: 할인율 > 1.0 (단기)
  - 영국 금리가 EUR보다 높음
  - FX 포워드에서 GBP 약세 예상

- **EUR-EONIA**: 기준 곡선
  - 할인율 0.99 ~ 1.0
  - 유로존 완화적 통화정책

- **USD-FedFunds**: 할인율 가장 낮음
  - 미국 높은 금리 반영
  - EUR 대비 USD 강세

- **EURIBOR-12M**: 12개월물 (장기 프리미엄)

## 4. 유럽 통화 간 관계 분석

### 4.1 EUR 주변 통화 역학

```
EUR 중심 통화 구조:
━━━━━━━━━━━━━━━━━━━━━

      GBP (1.2%)
        ↖
         ↖  스프레드 +100bp
           ↖
    CHF (0.8%) ── EUR (0.1%) ── USD (1.0%)
        ↗          기준            ↖
       ↗          유로존          ↗
    스프레드 +70bp               스프레드 +90bp
```

### 4.2 FX 포워드 가격 평가

**EUR 기준 포워드 계산:**
```
선도 환율 = 현물 × (1 + r_counter) / (1 + r_EUR)

EUR/GBP 포워드:
- GBP 금리 (1.2%) > EUR 금리 (0.1%)
- EUR 선도 가치 상승 (GBP 약세)

EUR/USD 포워드:
- USD 금리 (1.0%) > EUR 금리 (0.1%)
- EUR 선도 가치 상승 (USD 약세)
```

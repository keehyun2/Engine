# ======================================== #
# 시장 데이터 (Market Data) 파일 설명      #
# SABR 예제 - 2024년 3월 19일자          #
# ======================================== #
#
# [파일 형식: CSV]
# 날짜,데이터_종류,값
#
# [데이터 종류]
# 1. IR_SWAP/RATE: 금리 스왑 금리 (OIS)
# 2. MM/RATE: 머니 마켓 금리
# 3. OI_FUTURE/PRICE: 익일 인덱스 선물 가격
# 4. SWAPTION/RATE_NVOL: 스왑션 정규 변동성 (Normal Volatility)
#
# ======================================== #
# 1. OIS 금리 (Overnight Index Swap)      #
# ======================================== #
#
# 형식: IR_SWAP/RATE/USD/{SpotDelay}/{Index}/{Tenor}
#
# FedFunds OIS (2D/1D):
# 2024-03-19,IR_SWAP/RATE/USD/2D/1D/1M,  0.05339  # 1개월: 5.339%
# 2024-03-19,IR_SWAP/RATE/USD/2D/1D/1Y,  0.05069  # 1년:   5.069%
# 2024-03-19,IR_SWAP/RATE/USD/2D/1D/10Y, 0.03889  # 10년:  3.889%
# 2024-03-19,IR_SWAP/RATE/USD/2D/1D/20Y, 0.03846  # 20년:  3.846%
# 2024-03-19,IR_SWAP/RATE/USD/2D/1D/30Y, 0.03653  # 30년:  3.653%
#
# SOFR OIS (0D/1D):
# 2024-03-19,IR_SWAP/RATE/USD/SOFR/0D/1D/2Y,  0.04591  # SOFR 2년: 4.591%
# 2024-03-19,IR_SWAP/RATE/USD/SOFR/0D/1D/10Y, 0.03920  # SOFR 10년: 3.920%
#
# ======================================== #
# 2. 머니 마켓 금리 (Money Market)        #
# ======================================== #
#
# 형식: MM/RATE/{통화}/{Delay}/{만기}
#
# 2024-03-19,MM/RATE/USD/0D/1D,      0.05330  # FedFunds ON: 5.330%
# 2024-03-19,MM/RATE/USD/SOFR/0D/1D, 0.05310  # SOFR ON:     5.310%
#
# ======================================== #
# 3. SOFR 선물 가격                     #
# ======================================== #
#
# 형식: OI_FUTURE/PRICE/USD/{년월}/{거래소}/기간
#
# XCME:SRA = CME SOFR Futures (3개월)
#
# 2024-03-19,OI_FUTURE/PRICE/USD/2024-05/XCME:SRA/3M, 94.6575
# 2024-03-19,OI_FUTURE/PRICE/USD/2024-06/XCME:SRA/3M, 94.6650
# 2024-03-19,OI_FUTURE/PRICE/USD/2024-12/XCME:SRA/3M, 95.1200
# 2024-03-19,OI_FUTURE/PRICE/USD/2025-03/XCME:SRA/3M, 95.4100
#
# 선물 가격 → 금리 변환:
# 금리 = (100 - 가격)
# 예: 94.6575 → 금리 = 100 - 94.6575 = 5.3425%
#
# ======================================== #
# 4. SABR 스왑션 변동성                  #
# ======================================== #
#
# 형식: SWAPTION/RATE_NVOL/USD/SOFR/{스왑만기}/{옵션만기}/{종류}
#
# [변동성 종류]
# - ATM: At-The-Money (행사가 = 포워드 금리)
# - Smile/{Delta}: 행사가별 변동성 (스마일 곡선)
#   - Smile/-0.01: 포워드 대비 -1% (ITM)
#   - Smile/0.01:  포워드 대비 +1% (OTM)
#
# [예시: 10년 스왑 만기, 1년 옵션 만기]
# ATM (기준):
# 2024-03-19,SWAPTION/RATE_NVOL/USD/SOFR/10Y/1Y/ATM, 0.00928
#
# Smile (스마일 곡선):
# 2024-03-19,SWAPTION/RATE_NVOL/USD/SOFR/10Y/1Y/Smile/-0.04, 0.00943  # -4%
# 2024-03-19,SWAPTION/RATE_NVOL/USD/SOFR/10Y/1Y/Smile/-0.02, 0.00895  # -2%
# 2024-03-19,SWAPTION/RATE_NVOL/USD/SOFR/10Y/1Y/Smile/0.02,  0.01088  # +2%
# 2024-03-19,SWAPTION/RATE_NVOL/USD/SOFR/10Y/1Y/Smile/0.04,  0.01299  # +4%
#
# ======================================== #
# 5. Cap/Floor 변동성                   #
# ======================================== #
#
# 형식: CAPFLOOR/RATE_NVOL/USD/SOFR/{만기}/{Strike}
#
# Strike는 절대 수준 (예: 0.04 = 4%)
#
# ======================================== #
# [데이터 해석]
# ======================================== #
#
# 1. 금리 곡선 형태:
#    - 단기(1M): 5.34% (높음)
#    - 중기(10Y): 3.89%
#    - 장기(30Y): 3.65% (낮음)
#    - 정상적인 하향 곡선 (역전 가능성)
#
# 2. SOFR vs FedFunds:
#    - 거의 동일한 수준
#    - SOFR이 약간 낮은 경향
#
# 3. SABR 변동성:
#    - ATM 변동성: 약 0.8~1.0%
#    - 스마일 효과: OTM에서 변동성 상승
#    - 정규 변동성(Normal Vol) 사용 (음수 금리 가능)
#
# 4. 선물 가격:
#    - 2024년 5월: 94.66 → 금리 5.34%
#    - 2025년 3월: 95.41 → 금리 4.59%

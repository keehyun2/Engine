# [피싱(Fixing)이란?]
# - 변동금리(Floating Rate) 상품에서 적용되는 과거 확정 금리
# - 예: 3개월 LIBOR는 매영업일 확정(Fixing)되어 3개월 후 적용
# - FRA, 스왑 등 변동금리 상품에 필요
#
# [파일 형식]
# 날짜 인덱스 이름 값
#
# [예시]
# 20160201 USD-LIBOR-3M 0.0078000
# 20160201 USD-FedFunds 0.0045000
# 20160201 USD-Prime 0.0325000
#
# [주의]
# - 이 예제 파일은 비어있음 (피싱 데이터 불필요)
# - implyTodaysFixings="Y" 설정으로 오늘날짜 피싱은 자동 계산
# - 과거 날짜 피싱만 필요

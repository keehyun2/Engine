# SA-CCR 예제

## 개요
바젤 III 규제를 위한 **표준화된 카운터파티 신용 리스크 접근법(SA-CCR)** 계산을 보여줍니다.

## SA-CCR이란?

카운터파티 신용 리스크를 측정하는 표준화된 방법으로:
- Replacement Cost (현재 노출)
- Add-on (잠재적 미래 노출)
- Alpha Factor (배분 계수)
